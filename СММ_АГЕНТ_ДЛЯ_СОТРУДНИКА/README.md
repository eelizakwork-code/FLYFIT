# СММ-агент FlyFIT — пакет для сотрудника

## Что внутри

- `1_БЫСТРЫЙ_СТАРТ.md` — пошагово, как запустить агента за 5-10 минут.
- `2_СИСТЕМНЫЙ_ПРОМПТ_ДЛЯ_ВСТАВКИ.md` — готовый промпт (скопировать целиком).
- `3_ЧТО_ПРИКРЕПЛЯТЬ_В_ЧАТЕ.md` — какие файлы `@...` добавлять в разные типы задач.

## Коротко для сотрудника

1. Открой проект `CURSOR` в Cursor.
2. Создай своего кастомного агента и вставь промпт из `2_СИСТЕМНЫЙ_ПРОМПТ_ДЛЯ_ВСТАВКИ.md`.
3. Для постов в чат добавляй минимум:
   - `@О_БРЕНДЕ/Карточка_бренда_для_СММ_агента.md`
   - `@КОНТЕНТ/БАЗА/ПРАВИЛА_ПОСТОВ_для_агента.md`
4. Для тонкой стилистики добавляй `@КОНТЕНТ/БАЗА/100постов_тг.csv`.

## Важно

- Если пользователь не дал тему: агент сам выбирает тему и пишет почему выбрал.
- Если тема дана: пишет строго по ней.
- Вывод поста всегда в Markdown.
- Если пользователь просит пост без уточнений по площадке: агент сразу дает 4 версии в одном ответе:
  - Instagram
  - Telegram
  - Дзен
  - TikTok
- Все версии выдаются в отдельных копируемых блоках (код-блоках), чтобы можно было скопировать в один клик.
- Агент сохраняет единый смысл поста, но адаптирует под площадку:
  - длину текста,
  - хук/лид,
  - структуру,
  - CTA,
  - уместные хэштеги и оформление.
- Если пользователь дал артикулы, даты, акцию или точные формулировки — это приоритет над шаблонными правилами.
