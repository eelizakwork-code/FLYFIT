# Приложение допник ФФ

**Назначение:** материалы к дополнительному соглашению / приложению к договору возмездного оказания услуг с фулфилментом (Исполнитель — ИП Тепп К.А.; Заказчик — ИП Авезов Ю.Е., договор от 17.10.2024).

**Основной файл:** `Приложение_допник_порядок_работы_с_фулфилментом.md` — текст для вставки в допник; внизу файла — внутренняя памятка (таблицу в договор не включать).

**Источники при подготовке текста:** выдержки из договора, обратная связь сотрудников (сроки счетов, отчёты, фото отгрузок и брака, ТЗ на новые партии, день приёмки, поломка оборудования / простой ТС, неустойки), черновик сотрудников в `фф черновик.docx` (при необходимости сверить цифры с файлом).
