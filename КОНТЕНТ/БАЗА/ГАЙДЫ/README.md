# Гайды FlyFIT (Markdown)

Тексты сконвертированы из PDF для удобного поиска, правок и ссылок `@КОНТЕНТ/БАЗА/ГАЙДЫ/...` в Cursor.

| Файл | Исходный PDF |
|------|----------------|
| `Гайд_как_избавиться_от_отеков.md` | `Как избавиться от отеков.pdf` |
| `Чеклист_FLYFIT.md` | `Чек-лист FLYFIT.pdf` |
| `Вечерняя_рутина_FLYFIT.md` | `Вечерняя рутина FLYFIT.pdf` |

Повторная конвертация: скрипт `_convert_pdf_to_md.py` в этой папке (нужен PyMuPDF: `py -3 -m pip install pymupdf`).
