# Git и GitHub — шпаргалка

## Простая схема

1. **pull** — забрать с GitHub то, что сделали другие.
2. Поработала локально.
3. **add → commit → push** — отправить свои изменения.

---

## 7. Как подключиться к чужому проекту (команда)

Тебе дают ссылку на репозиторий. На компе в удобной папке в терминале:

```bash
git clone https://github.com/команда/проект.git
```

Потом в Cursor: **Open Folder** → выбрать скачанную папку. Дальше — те же **pull** / **add** / **commit** / **push** (если тебе выдали права на запись).

---

## 8. Где в Cursor это видно без команд

Слева есть значок ветки / **Source Control** (часто выглядит как развилка). Там можно:

- видеть изменённые файлы;
- написать сообщение коммита и нажать **Commit**;
- искать кнопки **Sync** / **Push** / **Pull** (зависит от версии).

Команды в терминале и кнопки делают **одно и то же** — выбирай, как удобнее.

---

## 9. Короткий словарь

| Слово | Простыми словами |
|--------|------------------|
| **Репозиторий (repo)** | Проект на GitHub + история версий |
| **Commit** | Снимок «как сейчас» + короткая подпись |
| **Push** | Отправить свои коммиты на GitHub |
| **Pull** | Скачать с GitHub новые коммиты к себе |
| **Clone** | Первый раз скачать весь проект с GitHub |
